LED Matrix RS Installer

Rust LED matrix HTTP API server for Raspberry Pi with RGB Matrix HAT.

Quick Install

User Install (no sudo)

Installs to ~/.local/bin

curl -fsSL https://brentwilkins.github.io/led-matrix-rs/install.sh | sh

System Install with systemd (recommended)

Installs to /usr/local/bin and sets up systemd service

curl -fsSL https://brentwilkins.github.io/led-matrix-rs/install.sh | sudo sh

System Install without systemd

curl -fsSL https://brentwilkins.github.io/led-matrix-rs/install.sh | sudo sh -s -- --no-systemd

Options

  • --version VERSION - Install specific version
  • --no-systemd - Skip systemd service installation
  • --verbose - Verbose output
  • --help - Show help message

Supported Hardware

  • Raspberry Pi Zero, Zero W/WH (ARMv6)
  • Raspberry Pi Zero 2 W (ARMv7)
  • Raspberry Pi 2/3/4 (ARMv7 or AArch64)
  • Adafruit RGB Matrix Bonnet (PID 3211)
  • 64x64 2mm pitch RGB LED panels
Note: GPIO access requires sudo. Even user installations need sudo led-matrix-rs to run.
